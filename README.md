# NeuroChat AI

An AI-powered Android chatbot application built using Grok API integration.

## Features

- AI-powered conversations
- Real-time API responses
- Modern Android UI
- Fast and responsive interface

## Technologies

- Java / Kotlin
- Android Studio
- Grok API
- Gradle
